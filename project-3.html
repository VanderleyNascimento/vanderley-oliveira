Sistema de Gerenciamento de Produtos

Este projeto é um Sistema de Gerenciamento de Produtos, desenvolvido para facilitar operações de CRUD (Create, Read, Update, Delete) em uma base de dados. Utilizando React para a criação da interface e Supabase como backend, o sistema oferece uma solução basica para gerenciar produtos.

Tecnologias:

  • React: Framework JavaScript para a construção da interface do usuário, com componentes dinâmicos e reutilizáveis.
  • Supabase: Backend como serviço, fornecendo uma base de dados segura e configurada com políticas de segurança a nível de linha (RLS).
  • SCSS: Pré-processador CSS utilizado para criar estilos mais organizados e escaláveis, melhorando a manutenção do código.
  • Tailwind CSS: Framework de utilitários CSS que facilita a criação de interfaces modernas e responsivas. Utilizando Tailwind, implementei um estilo inspirado no Material Design, garantindo uma estética limpa e consistente em toda a aplicação.

Funcionalidades:

  • Adição de Produtos: Formulário para adicionar novos produtos ao sistema, com validações e tratamento de erros.
  • Exclusão de Produtos: Funcionalidade que permite excluir produtos com suporte a autocomplete, facilitando a busca de itens específicos.
  • Gerenciamento de Erros Globais: Sistema centralizado para captura e exibição de erros, garantindo uma melhor experiência do usuário.

Conclusão:

O Sistema de Gerenciamento de Produtos demonstra a integração eficaz entre React e Supabase, resultando em uma aplicação web robusta e escalável. É uma solução ideal para ambientes que exigem operações de banco de dados basica.
